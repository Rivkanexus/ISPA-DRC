CSS overrides

Want to go all the way with customization? No problem! With CSS overrides you can make it even look like this. The sky is the limit. Just write your overrides at the bottom of the ‘style.css’ file in the ‘/static/css/’ directory. Look for the following line in this files and write your CSS below:

/*------------ CSS overrides below --------------*/
